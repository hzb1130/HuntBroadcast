# HuntBroadcast
Dual Language Mod Intro

## English
Damage & kill broadcast mod for The Long Dark.
Shows real-time attack stats, health status and kill notifications during hunting.

### Features
- Custom icon, color, display and fade time
- Toggle damage value, health and bleed display
- Hide last hit prompt optionally
- Filter valid damage sources, block fake corpse damage
- Separate alerts for player, animal and trap kills
- All settings configurable in game menu

### Screenshots

![Gameplay](https://github.com/hzb1130/HuntBroadcast/raw/main/1.png)

### Install
1. Install MelonLoader
2. Place DLL into `Mods` folder
3. Adjust options in game mod menu

## 中文
漫漫长夜狩猎播报模组，狩猎时实时展示伤害、血量与击杀信息

### 功能
- 自定义图标、文字颜色、显示及淡出时长
- 灵活开关伤害、血量、流血提示
- 可关闭最后一击弹窗
- 过滤有效伤害，屏蔽拾取尸体假伤害
- 区分玩家、野兽、陷阱三类击杀播报
- 全部参数游戏内可视化配置
